# P3 Moderator Panel Icons

This directory requires the following icon files for the PWA:

## Required Icons

1. **icon-192.png** (192x192 pixels)
   - Standard PWA icon size
   - Used for app icon on most devices

2. **icon-512.png** (512x512 pixels)
   - Large PWA icon size
   - Used for splash screens and high-resolution displays

## Design Guidelines

- Background color: #0d1b2a (dark navy blue to match theme)
- Use the P3 logo or moderator shield icon
- Ensure icons are square with no transparency issues
- Icons should be clearly visible at small sizes

## Icon Generation

You can generate these icons using:
- Figma/Sketch design tools
- Online PWA icon generators
- SVG to PNG conversion tools

Copy your PNG files to this directory with the exact filenames above.
